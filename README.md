# Real Time Chat Application with Authentication

A real-time chat application built with Node.js, Express, Socket.io, and React. Users can send private message with other users, send messages, and see messages instantly without refreshing the page.

Features

- Real-time messaging using WebSockets
- Display active users
- Responsive and user-friendly interface
- User authentication (email/password)

Tech Stack

- Backend: Node.js, Express, Socket.io
- Frontend: React
- Database: (TBD)
- Authentication: JWT

Installation

1. Clone the repository
2. Install backend and frontend dependencies
3. Set up .env file with database URI and JWT secret
4. Run backend and frontend servers

Usage

- Sign up or log in
- Send messages in real-time
- View active users and private chats
